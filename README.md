silex-user
==========

# Installation

Add this to your composer.json and then run `php composer update`:

    "require": {
        "silex/silex": "~1.0",
        "doctrine/dbal": "~2.3",
        "symfony/security": "~2.3",
        "symfony/twig-bridge": "~2.3",
        "Atelier1100/silex-user": "~2.3"
    }
